# 서비스공방

서비스공방은 빠른 마이크로 제품 제작을 위한 Agent Skills 모음입니다.

Use the skills in this order when the user asks for an end-to-end flow:

1. `micro-market-signal`
2. `micro-offer-spec`
3. `micro-build-loop`
4. `micro-launch-pack`

Use `micro-founder-loop` when the user asks for the full pipeline or is unsure where to start.

Before idea discovery, lock the search bounds with three short fields: buyer, arena, constraints. If any field is missing, ask one compact intake block and wait. Do not brainstorm from vague inputs.

Keep outputs concrete. Favor direct evidence, narrow V0 scope, verification commands, and launch assets the user can publish today.
